Перед Вами web-приложение - чат с возможностью видеозвонка, созданный с помощью технологии **webRTC**.

Перед запуском:
- заменить YOU_NAME_COMP в файле run.sh на своё имя компьютера
- sudo chown YOU_NAME_COMP: nginx_conf
- chmod +x run.sh
- sh run.sh
